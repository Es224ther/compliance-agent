# Sample Input Scenario

**Title:** Cross-border data transfer without DPA

**Description:**
A US-based SaaS startup collects personal data from EU residents and replicates it nightly to a cloud instance in India for analytics. No Data Processing Agreement (DPA) is in place, and users were not informed of the transfer in the privacy policy.

**Jurisdiction:** EU, US, IN
**Regulatory Domains:** GDPR, PDPB
